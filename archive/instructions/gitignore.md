# Gitignore Instructions

Download .gitignore files from the github gitignore repository
Url pattern is https://github.com/github/gitignore/raw/refs/heads/main/{library}.gitignore
download command options:
* wget -O {url} .gitigore
* curl -o .gitignore {url}
